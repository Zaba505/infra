gh issue create --title "story(self-hosted-personal-media-storage): define UX — delete own content" --label enhancement --body "$(cat <<'EOF'
Author the UX for a user deleting their own content, including the 30-day retention window for accident recovery.

**Capability:** self-hosted-personal-media-storage
**Trigger covered:** "A user deletes their own content."
**Business rule:** "30-day retention after deletion / departure."

## Scope
- User selects content and deletes it; it is removed from their view immediately.
- Within the 30-day window the user can restore the content.
- After 30 days the content is purged from the system (including off-site backup copies).
- Active shares to the deleted content cease to resolve once it is removed from view.

## Inputs
- Authenticated identity for the acting user.

## Outputs
- Deleted content is hidden immediately and purged after the retention window.

## Out of scope
- Operator-initiated termination (separate UX).
- User-initiated full account departure (separate UX).

## Done when
- UX doc authored via `define-user-experience` and human-reviewed.
EOF
)"
