These ADRs are scoped to the [Self-Hosted Application Platform](../_index.md) capability. Each one resolves one or more technical requirements from [tech-requirements.md](../tech-requirements.md). Cross-capability decisions live in `docs/content/r&d/adrs/`, not here.

Numbering is local to this capability and starts at 0001. Status lifecycle: `proposed` → `accepted` → (later) `superseded`.
